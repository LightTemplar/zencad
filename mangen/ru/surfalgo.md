# Классы поверхностей.

-----------------
## Классы поверхностей.
В ZenCad существуют следующие классы поверхностей:

* Face 
* Surface (NotReleased)

--------------------
## Поиск нормали
В точке поверхности, соответствующей параметрам _u_, _v_. По умолчанию _u_ и _v_ равны нулю.

Сигнатура:
```python
surf.normal(u=0,v=0)
```
