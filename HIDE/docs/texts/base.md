# Базовые объекты

## Point  
Объект типа точка. Является аргументом многих операций.
Может быть отображен.
```python
zencad.point(x, y, z)
zencad.pnt(x, y, z)
zencad.point([x,y,z])
```

Example:
```python
zencad.point(20, 20, 10)
```
