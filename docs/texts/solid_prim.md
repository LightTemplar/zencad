# CSG Примитивы

## Box  
{% highlight python %}
box(size = (x,y,z), center = True/**False**);
box(size = x, center = True/**False**);
{% endhighlight %}

## Sphere  
{% highlight python %}
sphere(r = radius)
{% endhighlight %}

## Cylinder  
{% highlight python %}
cylinder(h = height, r = radius, center = True/False);
{% endhighlight %}

## Cone  
{% highlight python %}
cone(h = height, r1 = botRadius, r2 = topRadius);
{% endhighlight %}
