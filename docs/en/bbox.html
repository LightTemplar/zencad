Bounding box

The bounding box is a minimal Ox, Oy, Oz-based box that describes the geometric shape shape.

shape.bbox

Construct a bounding box based on the shape.

Пример

shp = sphere(10)
bbox = shp.bbox()

Fields.

bbox.xmin
bbox.ymin
bbox.zmin
bbox.xmax
bbox.ymax
bbox.zmax

Methods.

bbox.xrange() bbox.yrange() bbox.zrange()

To Shape.

Пример

shp = sphere(10)
bbox = shp.bbox()
disp(bbox.shape())
